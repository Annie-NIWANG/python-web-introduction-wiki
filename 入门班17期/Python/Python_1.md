### Cloud migration
- 云迁移（Cloud migration）是指将数据、应用程序、服务或其他业务元素从一个公司的本地计算机或数据中心迁移到云计算环境的过程。
- 这个过程涉及到将业务的IT资源转移到云服务提供商（如AWS、Microsoft Azure或Google Cloud Platform）提供的远程服务器上。
