## HTML&CSS_Part2（22/10/2023）

### CSS的概念
- Cascading Style Sheet 级联样式表
- 表现HTML 或 XHTML 文件样式的计算机语言<br>
包括对字体、颜色、编剧、高度、宽度、背景图片、网页定位等设定

### CSS3基本选择器
- 标签选择器
- 类选择器
- ID选择器

### CSS导入方法
- 内部样式：html中直接添加
```html
<head>
<style>
      /* 选择器 {样式} */
      /* 给谁改样式 { 改什么样式} */
      /* key value */
</style>
</head>
```
